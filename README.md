# EduGrade Pro — Java Web Application

## Requirements
- Java 21+  (you have Java 25 - OK)
- Maven 3.6+ (you have Maven 3.9 - OK)

## Run (2 commands only)

Open terminal inside EduGradeWeb folder, then:

  mvn package -q
  java -jar target/EduGradePro.jar

Then open browser:
  http://localhost:8080

Stop: Ctrl + C
